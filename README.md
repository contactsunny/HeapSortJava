# Heap Sort

This is a simple Java program to demonstrate how to perform heap sort on a 
given array.

## Input

```shell script
Original array: [10, 7, 11, 30, 20, 38, 2, 45]
------------------------
```

## Output

```shell script
Sorted array: [2, 7, 10, 11, 20, 30, 38, 45]
------------------------
```
